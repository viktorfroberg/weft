# Changelog

weft is **pre-release** — no tagged version exists yet, so there's nothing to log here.

Once **v1.0.0** ships, per-release notes will live on GitHub:

**[github.com/viktorfroberg/weft/releases](https://github.com/viktorfroberg/weft/releases)**

Release notes are hand-curated when a tag is cut. The [`.github/release.yml`](.github/release.yml) config groups merged PRs into sections on the auto-draft; final notes are human-rewritten into prose. Same pattern as Helix, Alacritty, Ghostty.

For what's planned next, see [`docs/roadmap.md`](docs/roadmap.md).
